Dit is een sectie over de hele breedte van de pagina. width en height zijn niet opgegeven. Browser pakt de breedte van het hele scherm en de hoogte van de tekst.

Dit is een sectie over met width = 600px en height = 5 cm.
Kleur gaat verder dan tekst omdat height is opgegeven.

Je kan verschillende secties links uitlijnen en vervolgens langs elkaar zetten. Ieder van deze secties is 30% van het scherm.

Dit is een linksuitgelijnde sectie met width = 50% van moederelement. Moederelement is het scherm.

Dit is een rechtsuitgelijnde sectie met width = 40% van "moeder"-element. "Moeder"-element is het scherm.

Dit is een linksuitgelijnd artikel. De width = 50% van het "moeder-element. "Moeder"-element is de rechtsuitgelijnde sectie.

Hoogte van sectie past zich automatisch aan de inhoud aan.
Dit omdat height niet in de css is opgegeven.
